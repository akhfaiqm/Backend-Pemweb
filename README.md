# Backend Event Management System

Proyek ini adalah backend untuk sistem manajemen event. Proyek ini dibuat sebagai bahan pembelajaran mata kuliah Pemrograman Web 2 di Universitas Harkat Negeri Tegal.

## Deskripsi

Backend ini menangani logika server dan pengelolaan data untuk aplikasi event management, termasuk:
- manajemen event
- manajemen pengguna
- pendaftaran peserta
- otentikasi dan otorisasi

## Tujuan

- Membantu mahasiswa memahami konsep backend web
- Mengimplementasikan API untuk aplikasi event management
- Menyusun struktur proyek yang rapi dan mudah dipelajari

## Teknologi

Proyek ini menggunakan teknologi backend untuk membangun API dan pengelolaan data. Informasi detail tentang stack dan dependensi dapat dilihat di konfigurasi proyek.

## Cara Menjalankan

1. Clone repositori
2. Install dependensi
3. Jalankan server backend

Informasi lebih lengkap tentang langkah-langkah instalasi dan penggunaan dapat ditambahkan sesuai kebutuhan proyek.

## Kontribusi

Proyek ini dikembangkan sebagai proyek pembelajaran. Silakan tambahkan fitur dan pembaruan yang mendukung tujuan edukasi.
